# Student-Registration-Form

A simple, responsive student registration form built with HTML, CSS, and JavaScript, with client-side validation.

🔗 Live Demo : student-registration-form-psi-two.vercel.app

✨ Features
Clean, centered layout with dark theme
Responsive design
Validates required fields, email format, phone number (10 digits), and password length


🛠️ Built With

HTML5, CSS3, JavaScript (Vanilla)
